## 2024-05-22 - [Adding Context with Relative Time]
**Learning:** Terminal interfaces often sacrifice metadata for space, but time context ("2h ago") is critical for news relevance.
**Action:** Use short relative time formats (2h, 1d) to add high-value context without cluttering the TUI.

## 2024-05-23 - [Contextualizing Links with Domains]
**Learning:** In a news aggregator, knowing the source (domain) of a link drastically improves scanning efficiency and trust.
**Action:** Append the domain name (e.g., "(github.com)") to story titles, using dimmed styling to maintain hierarchy.
